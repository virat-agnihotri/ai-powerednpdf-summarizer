1. TKINTER
-It is python built in tool for building desktop tools .Its the graphical user interface (GUI).It converts python script into windows,button,menus and text fields.
-In tkinter, absolutely everything you see on the screen is called a widget.
-Every basic tkinter application follows the exact same lifecycle. If you miss one of these steps, your window either won't show up, or it will look completely empty.

import tkinter as tk

# Step 1: Create the main window (The Root)
root = tk.Tk()
root.title("My First GUI")
root.geometry("400x200") # Width x Height

# Step 2: Create a widget and place it
label = tk.Label(root, text="Hello, World!")
label.pack() # The geometry manager that places it on screen

# Step 3: Start the event loop
root.mainloop()
