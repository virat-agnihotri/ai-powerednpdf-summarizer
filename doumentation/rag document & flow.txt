EXAMPLE
Suppose user uploads:
Operating Systems.pdf
Backend receives PDF

STEP 1-
PyPDF2
or
pymupdf

text = extract_text(pdf)

STEP 2-
LLMs cannot handle huge documents efficiently.
Split into small pieces.

use:-
LangChain Text Splitter
chunk_size = 500
overlap = 100
Why Overlap?

Without overlap:

Chunk 1:
Virat is studying Artificial

Chunk 2:
Intelligence at college

Meaning breaks.

With overlap:

Chunk 1:
Virat is studying Artificial

Chunk 2:
studying Artificial Intelligence at college

Context remains.

STEP 4-
store vector in database

QUERY PROCESS

Suppose user asks:
What is CPU Scheduling?

Step 1: Query Embedding
Convert query:
What is CPU Scheduling?
↓
Vector

Step 2: Similarity Search
Search vector database.
FAISS finds closest chunks.
Example:
Chunk:
CPU Scheduling...
Similarity = 0.95
Chunk:
Memory Management...
Similarity = 0.42
Retrieve top chunks.
top_k = 3

Step 3: Build Prompt
Backend creates:
Answer only from context.
Context:
CPU Scheduling is...
Round Robin...
Priority Scheduling..
Question:
What is CPU Scheduling?

Step 4: Send to LLM
Could be:
GPT
Gemini
Claude
Local Llama
response = llm(prompt)

Step 5: Return Answer
Backend
   ↓
Frontend
   ↓
User
